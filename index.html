Wie ich meine erste HTML-Seite mit GitHub veröffentlicht habe

1. HTML-Datei erstellen

Ich habe eine einfache HTML-Datei namens index.html erstellt:

<!DOCTYPE html>
<html>
<head>
    <title>Meine erste Seite</title>
</head>
<body>
    <h1>Hallo Welt!</h1>
    <p>Meine erste veröffentlichte Webseite.</p>
</body>
</html>

2. GitHub Repository erstellen

Ich habe auf GitHub.com ein neues Repository angelegt:

  • Name: mein-erstes-projekt
  • Öffentlich (public)
  • Mit README.md initialisieren

3. Dateien hochladen

Mit diesen Befehlen habe ich meine Dateien hochgeladen:

git clone https://github.com/mein-name/mein-erstes-projekt.git
cd mein-erstes-projekt
git add index.html
git commit -m "Meine erste HTML-Seite"
git push origin main

4. GitHub Pages aktivieren

In den Repository-Einstellungen:

  1. Auf Settings klicken
  2. Links auf Pages gehen
  3. Branch main auswählen
  4. Speichern

Nach wenigen Minuten war meine Seite live unter:
https://mein-name.github.io/mein-erstes-projekt/

Erstellt mit ❤️ und GitHub Pages
